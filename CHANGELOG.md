## [1.0.0] - 2026-04-24

### Added
- Game đua xe audio cho người khiếm thị
- Google TTS đọc menu và thông tin race
- Điều khiển cảm ứng (nghiêng, vuốt, tap)
- Hỗ trợ TalkBack
- Multiplayer online qua server riêng
- Màn hình cài đặt với:
  - Tên người chơi
  - Âm lượng TTS
  - Âm lượng nhạc
  - Âm lượng hiệu ứng
- GitHub Actions CI/CD tự động build APK

### Features
- **Menu chính** (5 mục):
  1. Khởi động nhanh
  2. Thử thách thời gian
  3. Đua đơn
  4. Chơi multiplayer
  5. Cài đặt

- **Điều khiển**:
  - Vuốt trái/phải: Di chuyển menu
  - Vuốt lên: Chọn/Xác nhận
  - Vuốt xuống: Phanh
  - Nghiêng điện thoại: Rẽ trái/phải
  - Chạm đúp: Đề máy/Tắt máy

### Multiplayer Server
- Server: `https://topspeedapi-production.up.railway.app`
- Hỗ trợ tạo phòng, vào phòng, đua online
- Lưu tên người chơi

### Infrastructure
- GitHub Actions workflow cho auto-build
- Debug APK và Release APK tự động
- Documentation đầy đủ
